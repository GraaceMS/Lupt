
## Projeto 
- Aplicação capaz de Listar e Criar/Editar Jogadores de futebol e times.
## Comandos:
- back: npm start
- front: npm run dev
## Ferramentas :
- Nextjs, para o frontend 
- Nestjs + Prisma ORM para o backend
